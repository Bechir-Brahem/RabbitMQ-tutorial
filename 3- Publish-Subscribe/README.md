# setup intellij 
goto file -> project structure -> modules -> dependencies  
and add the ./lib folder as a dependency

# compile

```
./compile
```

# run

rabbitmq server must be running with user bechir and password bechir

## Subscriber

```
./runReceiveLogs
```

## Publisher

```
./runEmitLogs
```
